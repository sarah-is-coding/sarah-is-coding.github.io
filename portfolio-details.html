HomeCookHub: A Culinary Universe

HomeCookHub, my group senior design project, is a comprehensive, user-friendly recipe-sharing platform where culinary enthusiasts can explore, share, and plan their meals effortlessly. With a focus on simplicity and user engagement, the platform provides seamless navigation through a vast array of recipes, alongside an intelligent meal planner and automated grocery list generator. Users can explore and upload recipes, curate weekly/monthly meal plans according to dietary preferences, and generate precise grocery lists. Engaging features like recipe ratings, reviews, and user-generated content enrich the community-driven aspect of the platform, while personal dietary analytics, cloud storage, and mobile app integration amplify user experience and functionality.

Tech Stack

Frontend: React: Utilized for its component-based architecture, enhancing UI development, and ensuring a responsive and dynamic user interface.
Mobile: React Native: Adopted to provide a consistent user experience between the web platform and mobile application.
Backend: Node.js & Express.js: Employed to manage server-side logic, ensuring efficient data handling and interaction between the frontend and database.
Database: Firestore: Chosen for its scalability and proficient query capabilities, managing the vast array of user-generated recipe data and user profiles effectively.
Hosting & Deployment: Firebase Hosting: Secured for its reliable and swift web app hosting capabilities.
User Authentication: Firebase Authentication: Implemented for its straightforward and secure user sign-in options.
File Storage: Firebase Cloud Storage: Utilized to securely store and retrieve user-uploaded files such as images and videos.
Serverless Functions: Firebase Cloud Functions: Deployed to manage server-side operations seamlessly, without the need for an independent server infrastructure.
Version Control: GitHub: Leveraged for its robust version control, facilitating effective team collaboration and transparent work history display.
Testing: Jest & React Testing Library, Mocha/Chai: Engaged for unit and functional testing to ensure stability and reliability across UI and backend operations.

My Role
